HTML 작성규칙

1. 기본규칙

  • HTML은 해당 DTD의 명세에 맞게 작성하며, W3C Validation 통과를 원칙으로 한다.
    파이어폭스 부가기능 설치 및 사용안내
  • DTD를 제외한 모든 엘리먼트와 애트리뷰트는 소문자로 작성한다.
    잘못된 사용 ex) 〈P Class=""〉〈/P〉, 〈a href="" onClick="" onMouseOver=""〉
  • 애트리뷰트값은 큰따옴표("")로 묶는다.
    잘못된 사용 ex) 〈img src=‘test.gif’ width=‘150’ height=’25’ alt=‘테스트’〉
  • entity 코드 사용 필수 http://w3schools.com/tags/ref_entities.asp 참고
    잘못된 사용 ex) href=“board＆num=1” → board＆amp;num=1
  • html 페이지내에 자바스크립트 사용시 html 태그를 포함하고 있는 경우 //〈![CDATA[ , //]]〉 사용 필수.
    <script type="text/javascript"> //<![CDATA[ function test(){ console.log('hello'); } //]]> </script>
  • DTD XHTML 1.0을 사용중이므로 반드시 종료태그를 선언한다.
    주의해야할 종료태그 앨리먼트 : area, br, col, frame, hr, img, input, link, meta, param
  • 의미있는 객체를 구분하기 위하여 코드 그룹간 1줄씩 빈 줄을 만드는 것은 허용한다. 단, 빈 줄의 간격은 1줄을 초과하지 않는다.
  • 아이프레임(일반프레임) 사용시 title 속성 필수 사용

■ 들여쓰기규칙

* 마크업의 중첩이 깊어질때마다 자식 엘리먼트는 1탭씩 들여쓰고, 1탭의 공백은 4칸임.

드림위버와 에디터 플러스에서 Tab Size 설정하는 법

  • 드림위버 : Preferences > Category > Code Format > Indent ＆ Tab Size > [Indent width 1 Tabs, Tab Size 4]
  • 에디터플러스 : Tools > Preferences > Categories > Files > Settings and syntax > [Tab/Indent]

■ a 태그 사용시 주의사항

  • 새창을 포함하는 a 태그에는 반드시 target="_blank" 또는 title="xxx 페이지 새창으로 열림" 표시 필수
    ex) 〈a href="/front/pop/pop.html" target="_blank" title="테스트 페이지 새창으로 열림"〉
  • id 값 또는 javascript 값으로 이동할 경우 클릭시 다른곳으로 포커스가 이동하지 않도록 포커스 유지 처리가 되야 함

■ form 요소 사용시 주의사항

  • 테이블을 제외한 FORM 요소를 포함하는 경우에는 fieldset 구성필수
    ex) 검색, 이용약관(동의하기 포함) 그룹핑, 로그인 등
  • 기본적으로 명시적 label 사용.
    〈label for="id값"〉텍스트〈/label〉〈input id="id값" ...〉
  • label을 포함하지 않는 form 요소의 경우 title 값 필수
    ex) 〈select="" title="전화번호 국번선택"〉 〈input type="text" title="아이디를 입력해주세요"〉
  • label에서 하나 이상의 form 요소를 가지는 경우
    ex) 〈label for="id값"〉우편번호〈/label〉〈input type="text" id="id값" title="우편번호 앞자리"〉〈input type="text" title="우편번호 뒷자리"〉

■ 테이블 요소 사용시 주의사항

  • summary (테이블에 대한 요약정보), caption(테이블 제목) 사용 필수
    ex)summary"테이블에 대한 요약정보"
    caption="공지사항 게시판 리스트입니다."
  • th 태그에 scope="" 속성 필수 사용
    ex) th scope="col" th scope="row" th scope="colgroup" th scope="rowgroup(tbody에서는 하나의 rowgroup만 존재)"
